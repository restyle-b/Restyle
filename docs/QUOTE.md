# הצעת מחיר — אתר ומערכת Restyle

> מסמך זה הוא הצעת מחיר ללקוח. המחירים הם **טווחי הערכה מקובלים בשוק הישראלי** לפרילנסר/סטודיו
> קטן בפרויקט בסדר גודל זה — יש לעדכן לפי הסכמה סופית עם הלקוח לפני שליחה.

---

## רקע

הצעה לבניית הנוכחות הדיגיטלית של **Restyle** (מספרה), בשתי חלופות אפשריות לפי היקף הפרויקט
הרצוי — ממסך תדמית פרימיום ועד מערכת מלאה עם חנות, קורסים ואזור אישי.

---

## חלופה א' — דף עסקי פרימיום (Landing / תדמית)

אתר תדמית מעוצב ברמה גבוהה, בלי מסד נתונים או התחברות — מציג את המספרה ומפנה
את הלקוחות לאפליקציה הקיימת לקביעת תורים.

### כלול
- דף בית מעוצב פרימיום (השראת מותגים בינלאומיים מובילים בתחום) — תמונות גדולות, אפקטי גלילה
- עמודי תוכן: אודות, שירותים, גלריה, צור קשר
- טופס "השארת פרטים" (לידים) עם הגנת אנטי-ספאם ושליחת מייל/וואטסאפ להתראה
- כפתורי "קביעת תור" המפנים לאפליקציית Restyle הקיימת (App Store / Google Play / קישור)
- עיצוב מלא RTL ועברית, מותאם מובייל ודסקטופ
- SEO בסיסי (מטא-תגים, sitemap, מהירות טעינה)

### לא כלול
- חנות מקוונת, התחברות משתמשים, מסד נתונים, תשלומים, פאנל ניהול

### זמן פיתוח משוער
2–3 שבועות

### מחיר חד-פעמי (פיתוח)
**₪4,500 – ₪8,500** (תלוי בכמות עמודי תוכן, איכות/כמות תמונות, ועומק ההתאמה האישית לעיצוב)

### עלויות שוטפות (חודשי/שנתי, משולם ע"י הלקוח לספקים — לא לנו)
| שירות | עלות משוערת |
|---|---|
| אירוח (Vercel, שכבת Hobby) | ₪0/חודש — מספיק לאתר תדמית |
| דומיין | ₪50–150/שנה |
| **סה"כ שוטף** | **כ-₪5–15/חודש בממוצע** (בעיקר עלות דומיין מתחלקת לחודשים) |

## חלופה ב' — מערכת עסקית מלאה (Full Platform)

הרחבה מלאה הכוללת חנות, אקדמיה, אזור אישי, ניהול הזמנות ואינטגרציית תשלומים —
מערכת עם מסד נתונים, API, הרשמת משתמשים מאובטחת ופאנל ניהול.

### כלול
- כל מה שבחלופה א'
- **חנות**: קטלוג מוצרים, עגלת קניות, צ'קאאוט מלא
- **אקדמיה**: דפי קורסים, הרשמה לקורס
- **אזור אישי**: הרשמה/התחברות מאובטחת (Supabase Auth), היסטוריית הזמנות, גישה לקורסים שנרכשו
- **ניהול הזמנות**: מחזור חיים מלא (ממתין → בתשלום → שולם → במשלוח/הושלם → בוטל)
- **פאנל ניהול (Admin)**: ניהול מוצרים, הזמנות, קורסים, תוכן ומשתמשים
- **אינטגרציית סליקה**: חיבור לספק תשלומים ישראלי (Tranzila / HYP) לתשלום אונליין מאובטח
- מסד נתונים (PostgreSQL) + API מלא בצד שרת
- אבטחת מידע: הצפנת סיסמאות (מנוהל ע"י Supabase), בדיקת הרשאות בכל פעולה, הגנת מידע משתמשים (RLS)
- בדיקות אוטומטיות (unit + E2E) להבטחת אמינות לפני כל עדכון

### לא כלול
- עמלות סליקה בפועל (משולמות ע"י הלקוח לספק הסליקה לפי עסקה — לא חלק מהפיתוח)
- פתיחת חשבון/חוזה עם ספק הסליקה (Tranzila/HYP) — באחריות הלקוח, דורש עוסק מורשה/חברה
- צילום/הפקת תוכן מוצרים מקורי (אפשרי בתוספת תשלום אם נדרש)

### זמן פיתוח משוער
8–14 שבועות (תלוי בהיקף מוצרים/קורסים ובמספר סבבי משוב)

### מחיר חד-פעמי (פיתוח)
**₪35,000 – ₪65,000**

### עלויות שוטפות (חודשי/שנתי, משולם ע"י הלקוח לספקים — לא לנו)
| שירות | עלות משוערת |
|---|---|
| מסד נתונים + Auth (Supabase, שכבת Pro מומלצת לפרודקשן) | ~$25/חודש (≈₪95) |
| אירוח (Vercel, שכבת Pro מומלצת לתעבורה/תכונות מלאות) | ~$20/חודש (≈₪75) |
| אחסון תמונות (Cloudflare R2 — ללא עמלת egress) | כמה אגורות עד ~₪10/חודש בהיקף קטן-בינוני |
| ספק סליקה (Tranzila/HYP) | ~₪50–150/חודש דמי מנוי + ~1–3% עמלה לעסקה |
| דומיין | ₪50–150/שנה |
| **סה"כ שוטף (לפני עמלות סליקה לפי מחזור מכירות)** | **כ-₪220–400/חודש** |

> 💡 ניתן להתחיל בשכבות החינמיות (Vercel Hobby + Supabase Free) בזמן הרצת המערכת בפועל
> לפני עומסים גבוהים, ולשדרג רק כשצריך.

## הערה: איפה לאחסן את האתר — Vercel או Cloudflare?

האתר נבנה ב-Next.js עם פעולות שרת (server actions), הרשמת משתמשים והרשאות — טכנולוגיות
שמשתלבות באופן הטוב והפשוט ביותר עם **Vercel** (היצרן של Next.js), ולכן זו ההמלצה שלנו.

**Cloudflare כן משמש בפרויקט** — לאחסון תמונות (R2), ללא תלות באיפה מתארח האתר עצמו.

אם יש העדפה עסקית לאחסן את כל האתר ב-Cloudflare (למשל Cloudflare Pages/Workers):
- **אפשרי**, אך דורש שכבת התאמה (`@cloudflare/next-on-pages` או דומה) שלא תומכת ב-100% מהתכונות
  של Next.js (חלק מה-server actions וחלק מספריות Node.js עלולות לדרוש עבודה נוספת).
- משמעות בפועל: **זמן פיתוח/תחזוקה נוסף** ולעיתים פחות יציבות מהפתרון הטבעי (Vercel).
- מסד הנתונים (Supabase Postgres) לא תלוי בבחירת אירוח האתר ונשאר זהה בשתי האפשרויות.

**המלצתנו**: Vercel לאירוח האתר + Cloudflare R2 לתמונות (כמו שתוכנן) — השילוב הסטנדרטי
והבטוח ביותר לסטאק הזה. אם תרצה לעבור ל-Cloudflare מלא בכל זאת, נוכל להעריך זמן/עלות נפרדים.
