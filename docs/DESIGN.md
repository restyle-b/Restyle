# DESIGN — שפת עיצוב Restyle

מבוסס על ניתוח https://menspire.com (ההשראה) ומותאם למספרה Restyle בעברית/RTL.
מקור אמת לעיצוב ה-UI — לעדכן כשמתקבלות החלטות עיצוב.

---

## תחושה כללית (Mood)
פרימיום · גברי · נקי · חינוכי · סמכותי. לא "מספרה מזדמנת" אלא מותג עם עומק
(שירות + אקדמיה + מוצרים). מינימליזם עם תמונות גדולות שמדברות.

## פלטת צבעים
בהשראת השחור-לבן המתוחכם של menspire, עם צבע הדגשה אחד חם:
- **רקע ראשי כהה:** `#0E0E0E` (כמעט-שחור)
- **רקע משני:** `#1A1A1A`
- **טקסט על כהה:** `#FFFFFF` / `#E5E5E5`
- **רקע בהיר (סקציות חנות/תוכן):** `#FAFAF8` (לבן חמים)
- **טקסט על בהיר:** `#0E0E0E`
- **צבע הדגשה (accent):** אפור בהיר שטוח — `#E5E5E5` (hover: `#CFCFCF`), נדגם מתמונת לוגו הייחוס של המשתמש. הוחלף מזהב/ברונזה (`#C9A227`/`#B08D57`) המקוריים לפי בקשת המשתמש. נוסה גם מרקם בטון מחוספס (SVG noise) על אלמנטי ה-accent ונוסה טקסט לבן בכפתורים — **שניהם הוסרו** לפי משוב המשתמש; הצבע הסופי הוא אפור שטוח, ללא מרקם, עם `text-ink` על גביו.
- **גבולות/קווים עדינים:** `#2A2A2A` (כהה) / `#E8E6E0` (בהיר)

> ניגודיות גבוהה היא הליבה. שימוש מדוד מאוד בצבע ההדגשה (כפתורים/קישורים/הובר).

## טיפוגרפיה
- **כותרות וגוף טקסט:** **Assistant** (עברית/אנגלית) ו-**Cairo** (ערבית — Assistant לא תומך בה). מינימליסטי-נקי בהשראת orhayoon.com — אין גופן סריף נפרד לכותרות; היררכיה לפי משקל/גודל בלבד.
- **היררכיה:** הבדל דרמטי בגודל בין H1 ענק (clamp ~2.5–4.5rem) לטקסט גוף — כמו menspire.
- RTL מלא; מספרים/מחירים בפורמט עברי-ישראלי (₪).

## עקרונות פריסה (Layout)
- **Hero מסך-מלא** בדף הבית: תמונת רקע כהה + כותרת מיצוב + CTA בולט.
- רוחב מלא לסקציות תמונה; קונטיינר ממורכז (max-w ~1280px) לתוכן.
- מרווחים נדיבים (whitespace), גריד נקי, פינות מעט מעוגלות.
- אנימציות עדינות בלבד: fade/slide on-scroll, hover מעודן. ללא עומס.

## ניווט (מיפוי menspire → Restyle)
| menspire | Restyle (עברית) |
|----------|------------------|
| Learn More | אודות (הסיפור, צוות, קריירה) |
| Academy | אקדמיה (קורסים, הדרכה 1:1) |
| Shop | חנות (טיפוח, כלים, מבצעים) |
| Locations | מיקום ושעות |
| Book Now | **קביעת תור** (CTA ראשי → אפליקציית Restyle) |
| Help | צור קשר / עזרה |

> **קביעת תור** אינה מנוהלת באתר — קיימת אפליקציית Restyle. ה-CTA מפנה לחנויות האפליקציות / קישור web.

**CTA חוזרים לאורך הדף:** "קביעת תור" (→ אפליקציה) + "לחנות".

## סקציות דף הבית (סדר מומלץ)
1. **Hero** — תמונה גדולה + טאגליין + "קביעת תור".
2. **שירותים** — תצוגת שירותי המספרה (תספורת/עיצוב/זקן...).
3. **הזמנת תור** — בלוק בולט עם CTA.
4. **מוצרים נבחרים** — רשת best-sellers + "לחנות".
5. **אקדמיה** — הצגת קורס מוביל + "לכל הקורסים".
6. **אודות תקציר** — סיפור המותג + תמונה.
7. **גלריה / לפני-אחרי**.
8. **המלצות לקוחות**.
9. **מיקום, שעות, צור קשר + Footer**.

## קומפוננטות מפתח (shadcn/ui כבסיס)
- Button (primary כהה/accent, ghost), Card מוצר/קורס, Navbar שקוף→מלא בגלילה,
  Hero, Section wrapper, Gallery grid, Testimonial, Footer רב-עמודות.

## נגישות
ניגודיות AA לפחות, focus states ברורים, alt לכל תמונה, ניווט מקלדת מלא.

- לוגו Restyle (לקביעת צבע ההדגשה הסופי).
- 5–10 תמונות איכותיות (מספרה/עבודות/צוות) ל-Hero והגלריה.
- אם יש מיתוג קיים (צבעים/פונט) — לשלוח.
> בלי אלה אעבוד עם placeholders ופלטה שמוצעת כאן, וקל להחליף בהמשך.
