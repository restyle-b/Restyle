# Stage 1 — השלמת חלופה א' (אתר תדמית)

> מסמך תכנון. נוצר 2026-06-18. מקור אמת להתקדמות: `ROADMAP.md`.

## הגדרה
**Stage 1 = חלופה א'** (ראה `docs/QUOTE.md`): אתר תדמית פרימיום בלי חנות ובלי הרשמת
משתמשים. מטרת השלב: שהאתר החי יציג **בדיוק** את התכולה הנמכרת — לא פחות (נגישות,
אקדמיה, מיקום) ולא יותר (חנות, התחברות).

## פערים שנסגרים
1. **נגישות** — חובה לפי תקנות שוויון זכויות לאנשים עם מוגבלות (ת"י 5568 / WCAG 2.0 AA).
   לא קיים כלל. כולל: תפריט נגישות, קישור "דלג לתוכן", עמוד הצהרת נגישות.
2. **עמוד אקדמיה** (`/academy`) — מקושר מדף הבית/ניווט אך מחזיר 404. תוכן בלבד (ללא הרשמה — זו חלופה ב').
3. **עמוד מיקום ושעות** (`/locations`) — מקושר מהניווט אך מחזיר 404.
4. **הסתרת שאריות חלופה ב'** מה-UI (הקוד נשאר בריפו): קישור התחברות/אזור אישי,
   כפתור+קישור חנות, אזכורי "חנות" בכותרות/מטא-דאטה.

## מודל נתונים / קונפיג (אין Prisma — סטטי)
- `siteConfig` (`src/lib/config.ts`): `contact` אמיתי, `hours` (מערך), `accessibility`
  (`coordinatorName`, `phone`, `email`, `lastUpdated`).
- `navLinks`: הסרת `/shop`, הוספת `/gallery`.
- `src/lib/academy-data.ts` (חדש): מערך קורסים `{ slug, name, description, duration, level }`.

## חוזה Server Actions
אין חדשים. נגישות = client-side בלבד (localStorage). אקדמיה/מיקום = RSC סטטי.

| רכיב | קובץ | סוג |
|---|---|---|

פונט קריא, עצירת אנימציות, סמן גדול, איפוס. שמירה ב-localStorage.

## אבטחה
ווידג'ט client-only, ללא נתוני משתמש, ללא `innerHTML`/`dangerouslySetInnerHTML`,
ללא env/סודות. נתיבי `/login` נשארים נגישים ב-URL אך מוסתרים מהניווט; `middleware`
עדיין מגן על `/account`. אין רגרסיית אבטחה.
