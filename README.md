# Restyle 🪒

אתר פרימיום למספרה — חנות מוצרים, אקדמיה, אזור אישי מאובטח, ניהול הזמנות וממשק ניהול מתקדם.
בהשראת [menspire.com](https://menspire.com). ממשק בעברית (RTL).

## סטאק
Next.js 15 · TypeScript · Tailwind + shadcn/ui · Supabase (Postgres + Auth) · Prisma · Cloudflare R2 · Vercel
